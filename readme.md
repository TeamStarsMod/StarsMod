# StarsMod
## Stars是一个在Minecraft中提供简易星系的Mod
## 目前支持英文和中文语言，如果您想要帮助我们添加您自己的语言，欢迎克隆仓库后在src/main/resources/assets/starsmod/lang添加您自己的语言
## 此Mod使用Fabric模组加载器，当前唯一的开发版本为1.20.1
## 欢迎提交PR帮助我们开发!
